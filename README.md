# LearnAbility | Life-Skills Helper 🧩

**LearnAbility** is a comprehensive, real-time Learning Management System (LMS) designed specifically for neurodivergent students at the **NeuroSamarth NGO**. The platform digitizes essential life-skills training through structured, visual, and interactive modules.

## 🚀 Live Demo
https://life-skills-helper.vercel.app/

## 📖 How to Use the App

### 1. The Teacher Dashboard (Control Center)
Password is given to NGO for security access
* **Student Overview:** View a real-time list of all students currently enrolled in the system.
* **Assigning Tasks:** Click the dropdown menu next to a student's name to select a task (e.g., *Road Crossing*, *Handwashing*, or *Grocery Sorting*).
* **Live Update:** Once a task is selected, the system automatically pushes that specific module to the student's device without requiring a page refresh.
* **Tracking Progress:** Monitor the "Attempts" and "Success" columns to see how many times a student has tried a task and their current progress percentage.

### 2. The Student Portal (Learning Interface)
* **Automatic Sync:** The student simply keeps their portal open. When the teacher assigns a new task, the screen changes instantly to the new activity.
* **Interactive Steps:** * **Visuals:** Watch the Lottie or GIF animations showing the action.
    * **Audio:** Listen to the voice-guided instructions for each step.
    * **Action:** Click 'Next' once a step is completed to move forward.
* **Cognitive Flexibility:** In the *Road Crossing* module, follow the "Reverse Logic" (Red = Go) to complete the comprehension test.

### 3. Task Completion
* Once a student finishes all steps in a module, a "Success" message appears.
* The data is instantly sent back to the Teacher Dashboard so the instructor knows the task is finished.

## ✨ Key Features
* **Individualized Learning:** Each student has a unique task assignment scoped to their ID.
* **15+ Learning Modules:** Detailed sequences for Hygiene, Social Skills, Dressing, and Household Safety.
* **Accessibility First:** Integrated voice-overs and high-contrast visuals for students with learning difficulties.

## 🛠️ Tech Stack
* **Frontend:** React.js, Tailwind CSS
* **Backend & Database:** Supabase (PostgreSQL)
* **Real-time Logic:** Supabase Real-time Subscriptions
* **Animations:** LottieFiles & Giphy API
* **Development:** AI-Assisted Architecture via Lovable

## 📈 Impact
This project provides a data-driven approach to special education at **NeuroSamarth NGO**, replacing manual tracking with a centralized digital solution to foster student independence.
